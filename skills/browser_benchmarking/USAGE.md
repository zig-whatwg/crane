# Browser Benchmarking Skill Usage

## When to Use

**Load this skill when benchmarking implementation performance.**

Automatically loaded when:
- Benchmarking against browser implementations
- Comparing performance (Chrome, Firefox, Safari)
- Identifying optimization opportunities
- Measuring performance regressions
- Setting performance targets

## What It Provides

1. **Context-aware benchmarking** - Adapts to spec (URL, Encoding, Streams, etc.)
2. **Browser comparison** - How to benchmark against browsers
3. **Performance targets** - Realistic goals based on browser implementations
4. **Optimization patterns** - Spec-specific optimizations
5. **Benchmark tools** - Zig Timer, browser DevTools, datasets

## Key Principle

**Measure before optimizing.** Always benchmark current implementation before optimization.
