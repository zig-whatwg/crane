# WHATWG Skill Usage

## When to Use

**Load this skill when implementing any WHATWG specification.**

Automatically loaded when:
- Working in `src/url/`, `src/encoding/`, `src/streams/`, `src/infra/`, etc.
- Implementing features from WHATWG specs (URL, Encoding, Streams, Infra, etc.)
- Need to read WHATWG spec algorithms
- Mapping spec concepts to Zig implementations
- Understanding spec state machines or parsers

## What It Provides

Complete workflow from WHATWG spec to Zig implementation:
1. **Spec Navigation** - Find and read complete algorithms (never fragments)
2. **Context Detection** - Identifies which spec from file paths
3. **Type Mapping** - Map spec types to idiomatic Zig types
4. **Implementation Patterns** - Numbered comments matching spec steps
5. **Documentation** - Spec reference format
